# TimezoneTelegramBot
## Linux setup
Create **.env** file in root project directory with you **token**:
```env
TELEGRAM_BOT_TOKEN=YOU_TOKEN_FROM_FATHER_BOT
```

## Example setup:
```bash
git clone https://github.com/Straverk/TimezoneTelegramBot
cd TimezoneTelegramBot
touch .env
echo TELEGRAM_BOT_TOKEN=YOUR_TOKEN > .env
```
